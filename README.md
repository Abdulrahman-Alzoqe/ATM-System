# ATM Simulation System (C++) 🏧

A simple, interactive console-based ATM program built in **C++** that manages user accounts, handles cash transactions, and saves data to a local text file.

---

## 🚀 Main Features
* **Secure Login:** Access via Account Number and PIN Code.
* **Cash Withdrawal:** Supports both Quick Withdraw (predefined amounts) and Normal Withdraw (multiples of 5).
* **Cash Deposit:** Add funds instantly to your account balance.
* **Check Balance:** View your current live balance.
* **Data Persistence:** Automatically reads and writes all client data to a `Clients.txt` file.

---

## 📁 Files Included
* `Clients.txt` - The local database for storing client accounts and balances.

---

## 🛠️ How to Run
1. Make sure Clients.txt is in the same directory as the source code.
